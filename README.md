# Insurance Analysis Dashboard

## Overview
This Power BI project analyzes insurance customer data, premium collections, claims performance, and policy distribution. The dashboard helps insurance providers understand customer behavior and optimize business performance.

## Business Problem
Insurance companies need visibility into policy sales, claim trends, and customer segmentation to improve profitability and customer satisfaction.

## Tools & Technologies
- Power BI
- DAX
- Power Query
- Excel
- Data Modeling

## Key Performance Indicators (KPIs)
- Total Premium Amount
- Total Claims Amount
- Number of Policies
- Claim Settlement Ratio
- Customer Count
- Average Premium per Customer
- Active Policies

## Dashboard Features
- Premium Collection Analysis
- Claim Amount Analysis
- Customer Segmentation
- Policy Type Distribution
- Regional Performance Analysis
- Customer Demographics Dashboard
- Revenue Trends

## Key Business Insights
- Identified top-performing insurance products.
- Analyzed customer demographics driving revenue.
- Evaluated claim patterns across policy categories.
- Tracked premium growth trends over time.
- Highlighted regions contributing the highest business volume.
